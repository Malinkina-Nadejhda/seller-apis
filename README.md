# seller_py

Скрипт для автоматической синхронизации остатков и цен с сайта поставщика на маркетплейс Ozon.

## Что делает:

Скачивает архив с актуальными остатками и ценами с официального сайта [Timeworld](https://timeworld.ru). Собирает список артикулов товаров, которые размещены в личном кабинете продавца. Сравнивает оставшиеся товары у продавца с наличием у поставщика, если товара нет у поставщика, его количество на Ozon автоматически станет равным нулю. Обновляет стоимость товаров и остатков, и очищает цены от лишних символов и копеек.
