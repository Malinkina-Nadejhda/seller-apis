# seller.py

Скрипт для автоматической синхронизации остатков и цен с сайта поставщика на маркетплейс Ozon.

## Что делает:

Скачивает архив с актуальными остатками и ценами с официального сайта [Timeworld](https://timeworld.ru). Собирает список артикулов товаров из магазина продавца. Формирует остатки и новые цены для товаров на основе анализа остатков магазина поставщика. Конвертирует цены в чистые целые числа и выгружает обновленные цены и остатки в магазин Ozon.
